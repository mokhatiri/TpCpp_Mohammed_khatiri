# TP : Fondements des langages de programmation 2 (BCS 221 - E1) Advanced C/C++

## Introduction

This repository contains the course exercises and answers for the "Fondements des langages de programmation 2" (BCS 221 - E1) which focuses on advanced C/C++ programming.

## Repository Structure

- each folder is a seperate tp.
- you'll find the tp instructions in a pdf file within the folder in question
- the folders are structured as follow:
  - README.md (a readme file for each repository)
  - main.cpp (the main code for the tps)
  - [smth].cpp (additional cpp files related to the tp)
  - th file may also include executables for different exercices

## Installation

To use the materials in this repository, follow these steps:

1. **Clone the repository**:
   ```sh
   git clone https://github.com/mokhatiri/TpCpp_Mohammed_khatiri.git
   ```
2. **Navigate to the repository directory**:
   ```sh
   cd TpCpp_Mohammed_khatiri
   ```

## Usage

You can compile and run the C/C++ programs provided in the each tp directory. For example, to compile and run a program:

1. **Navigate to the tp 'k' directory using *md* **:
   ```sh
   cd TPk_KHATIRI_MOHAMMED
   ```
2. **Compile the program**:
   ```sh
   g++ -o program program_name.cpp
   ```
3. **Run the program**:
   ```sh
   ./program_name
   ```

(better read the readme file in each tp for better understanding)
